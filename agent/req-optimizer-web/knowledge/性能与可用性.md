# 前端/接口 性能与可用性指标参考

## Web Vitals（移动端优先）
- LCP ≤ 2.5s（好）/ 4.0s（差）
- INP ≤ 200ms（好）/ 500ms（差）
- CLS ≤ 0.1
- 首屏 TTI ≤ 3s（4G）

## 后端接口
- P95 ≤ 500ms 为达标，P99 ≤ 1s
- 错误率（5xx）≤ 0.1%
- 单实例 QPS 评估 = 日活 × 人均请求数 / 86400 × 安全系数 5

## 高可用
- SLA：核心链路 99.95%，普通 99.9%
- 多可用区部署 + 健康检查 + 自动故障转移
- 关键接口必须有降级方案（如登录失败时跳静态页）

## 兼容性默认值
- 浏览器：Chrome / Edge / Safari 最近 2 个大版本，Firefox ESR
- 移动端：iOS ≥ 14，Android ≥ 8
- 屏幕：≥ 360px 宽度，需要测 360 / 375 / 414 / 768 / 1280 五档

## 可观测性
- 日志（结构化 JSON）+ 指标（Prometheus）+ 链路追踪（OpenTelemetry）三件套
- 错误必须能在 5 分钟内告警到值班群
