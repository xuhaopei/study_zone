# 合规与隐私 Checklist

## 个人信息保护（PIPL / GDPR 通用）
- 收集前必须明示"采集字段、用途、存储期限"，并取得用户同意
- 用户有 4 项基本权利：查询 / 更正 / 删除 / 导出
- 数据出境（含使用境外云服务）需做安全评估或签订 SCC

## 最小化原则
- 仅收集业务所需的最小字段，禁止"以备将来用"
- 手机号、身份证号、生物特征属于敏感个人信息，需加密存储 + 单独同意

## 第三方登录（OAuth）
- 必须在隐私政策中列出所有第三方 SDK 名称、收集字段
- 默认不勾选"同意"，且第三方授权失败要给出可读错误

## 日志与审计
- 关键操作（登录、改密、改手机、删除账号）必须留痕，保留 ≥ 180 天
- 日志中严禁记录密码、完整身份证号、银行卡号

## 未成年人
- 14 岁以下需监护人单独同意
- 涉及未成年人时禁用个性化推荐、广告定向
